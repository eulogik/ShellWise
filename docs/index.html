Features
Everything you need.
Nothing you don't.
Built for developers who want the power of AI without the friction.
🧠

Smart Command Routing

Known commands run instantly. Natural language goes to AI. Zero friction.

🔒

Local Safety Classifier

Independently classifies every command — even AI-generated ones. Dangerous commands are blocked locally.

⚡

Response Caching

Repeated queries served instantly from cache. Smart TTL keeps responses fresh.

🌐

Rich Context

Knows your OS, git branch, package manager, directory contents — generates better commands.

🛡️

TUI Detection

Automatically detects interactive apps (vim, ssh, tmux) and offers to hand off to your shell.

📦

Fully Offline

Bundled llama-server binary. No cloud, no API keys, no data leaves your machine.

How it works
Three steps. That's it.
No configuration, no API keys, no learning curve.

Install

pip install shellwise

One command, no extras. Downloads the model on first run (~400 MB, one time).

Type naturally

Say what you want in plain English. shellwise translates it to the right command.

Review & execute

See the command, confirm if needed, done. Everything stays in your terminal.

Safety
Your terminal. Your rules.
shellwise classifies every command independently — even if the AI suggests something dangerous.
Type Behaviour Example
read Runs immediately, streams output ls, cat, grep, du
write Shows impact, asks for confirmation mkdir, cp, mv, touch
critical Locally blocked — never executes rm -rf /, dd, mkfs
Get started
Up and running in 30 seconds
# One command, no extras needed — bundled binary included
pip install shellwise
# First run downloads the model automatically (~400 MB, one time)
sw

Prefer a different backend?

# GPU (CUDA)
CMAKE_ARGS="-DGGML_CUDA=on" pip install "shellwise[gpu]"
# Or Ollama
ollama pull qwen2:0.5b && sw
sw
